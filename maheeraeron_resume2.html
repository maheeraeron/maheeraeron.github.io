Projects

Mobile School Volunteer Application (2017-2018)

Worked on Android/iOS app w/ Angular and Ionic Framework to make volunteer application for JHS.
Typescript & HTML/CSS for front-end; C#, ASP.Net based WebAPI for middle-tier; T-SQL for server.

3D racing video game for technology student association (2017-2018)

Programmed vehicle driver functionality and physics using C++, GUI using blueprints all
in Unreal Engine 4. 3DS Max for 3D car models + in-game environments, and Adobe Illustrator
for vector graphics.

3D space rpg video game for technology student association (2018-2019)

Physics and root motion based third person character based off bi-pedal humanoid movement made
entirely using C++ in Unreal Engine 4. Objects and environments created in 3DS Max.
Dialogue and quest system plugin developed in C++ supporting voiceover events and
character interaction.

HokieScheduler Student Schedule Application

Android Application that enables users to easily create and optimize a visualization of their
schedules for the upcoming semester at Virginia Tech. HokieScheduler offers a modern
user friendly interface for students to create a practical schedule and make valuable decisions
on their courses.
